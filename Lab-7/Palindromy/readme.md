# Palindoromy– Lab 7

Prosty program, który sprawdza czy podany tekst jest palindromem
## Opis działania

Program prosi użttkownika o tekst, który sprawdza on co do tego czy jest on palindromem

### Kompilacja

```
gcc -Wall -Wextra main.c -o Palindoromy
```

### Uruchomienie

Linux/macOS:

```
./Palindoromy
```

Windows:

```
chcp 65001 
.\Palindoromy.exe
```

### Dane testowe

#### Dane 1:
```
Podaj tekst do sprawdzenia czy nie jest palindromem: 
Anna
Jest palindromem
```

#### Dane 2
```
Podaj tekst do sprawdzenia czy nie jest palindromem: 
Kostka
Nie jest palindromem
```

#### Dane 3
```
Podaj tekst do sprawdzenia czy nie jest palindromem: 
Czekolada
Nie jest palindromem
```

#### Dane 4
```
Podaj tekst do sprawdzenia czy nie jest palindromem: 
sUs          
Jest palindromem
```
